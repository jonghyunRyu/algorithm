# 🧠 Algorithm Practice Repository

알고리즘 문제 해결 능력을 기르기 위해 풀이한 문제들을 정리해두는 레포지토리입니다.
코딩 테스트 대비와 자료구조·알고리즘 이해를 목표로 합니다.

---

## 📌 목적 (Purpose)

* 알고리즘 문제 해결 능력 향상
* 자주 등장하는 패턴 및 자료구조 정리
* 풀이 과정과 사고 흐름 기록
* 코딩 테스트 및 실무 대비

---

## 🛠 사용 언어 (Languages)

* JavaScript (ES6+)
* Java

필요 시 다른 언어 풀이도 추가할 수 있습니다.

---

---

## 🚀 목표

* 기본 자료구조 및 알고리즘 완전 이해
* 구현 문제에 대한 속도 및 정확도 향상
* 문제를 보고 적절한 자료구조를 빠르게 선택할 수 있는 능력 기르기
