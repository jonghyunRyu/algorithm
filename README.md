* **문제 링크 또는 출처 명시**

## 🧩 문제 풀이 형식 (Example)

```js
// 문제: 피보나치 수열
// 접근 방식: DP + 메모이제이션
// 시간복잡도: O(n)
// 공간복잡도: O(n)

const fibo = (n, memo = {}) => {
  if (n <= 1) return n;
  if (memo[n]) return memo[n];

  memo[n] = fibo(n - 1, memo) + fibo(n - 2, memo);
  return memo[n];
};
```
